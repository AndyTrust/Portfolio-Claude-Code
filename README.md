# 📊 Portfolio Intelligence Dashboard

Dashboard personale di analisi degli investimenti — costruita con Claude in Cowork mode.

## 🌐 Live Demo

**[→ Apri il Dashboard](https://andytrust.github.io/Portfolio-Claude-Code/)**

## Cosa contiene

- **Intelligence** — News aggiornate, sentiment aggregato, market pulse, Polymarket
- **Geopolitica** — Risk monitor globale, conflitti attivi, banche centrali, commodity, calendario macro
- **Reports** — Intelligence reports giornalieri in formato markdown renderizzati
- **Money Follow** — Tracking SEC 13F (fondi istituzionali), Form 4 (insider), STOCK Act (politici USA)
- **Fondi** — Heatmap fondi × titoli, stock cards con top buyer/seller/insider, analyst targets
- **Screener** — Screener titoli per settore con link OpenInsider

## Fonti dati

| Fonte | Tipo |
|-------|------|
| SEC EDGAR 13F | Fondi istituzionali (trimestrali) |
| OpenInsider + Form 4 | Insider activity (2gg) |
| Capitol Trades | STOCK Act — politici USA |
| WhaleWisdom | Analisi fondi 13F |
| Bloomberg / Reuters / FT | News e analisi |

## Note

- Dashboard statico HTML/JS — nessun backend, nessuna API key esposta
- I dati sono aggiornati manualmente ogni 3-7 giorni
- Non costituisce consulenza finanziaria
